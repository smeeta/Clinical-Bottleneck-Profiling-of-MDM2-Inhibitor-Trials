# MDM2-Inhibitor-Clinical-Bottleneck-Analysis-AML

## Overview
This repository contains R scripts, datasets, and reports for analyzing **MDM2 inhibitor clinical trials in Acute Myeloid Leukemia (AML)**.  
The analyses focus on therapeutic bottlenecks, genotype associations, and survival outcomes.

## Repository Structure


## Analysis Modules

1. **Clinical Bottleneck Analysis**
   - Heatmaps and frequency plots for PK, PD, and toxicity bottlenecks across trials.
2. **Genotype Analysis (MDM2 & TP53)**
   - Investigates relationships between MDM2 pathway alterations and TP53 genotype status in AML.
3. **Survival Analysis**
   - XXXXX and functional impact of MDM2/TP53 on patient outcomes.
